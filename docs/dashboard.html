DASHBOARD

O Dashboard HackInSDN é uma plataforma web para experimentação em redes e cibersegurança utilizando como Kubernetes como infraestrutura de execução. O Dashboard HackInSDN está totalmente integrado à arquitetura HackInSDN, permitindo uso dos módulos de detecção de anomalias com IA, contenção de ataques, simulação de ameaças e monitoramento de redes.

Mais informações: https://github.com/hackinsdn/dashboard
