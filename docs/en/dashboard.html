DASHBOARD

The HackInSDN Dashboard is a web platform for experimentation on cybersecurity and networking using Kubernetes cluster. The Dashboard is fully integrated with the HackInSDN architecture, allowing the utilization of all other components such as AI-based anoumaly detection, attack containment, adversarial traffic generator and security monitoring.
